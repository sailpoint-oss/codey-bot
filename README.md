# Codey-Bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that An automated developer community helper

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Docker

```sh
# 1. Build container
docker build -t Codey-Bot .

# 2. Start container
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> Codey-Bot
```
